El sistema Museo Asistente permite realizar consultas sobre hechos y eventos históricos del Museo de Realicó utilizando un motor de búsqueda híbrido, que combina análisis léxico y vectorial.

Cuando un usuario ingresa una pregunta en el chat web, el backend desarrollado en Node.js la procesa y la envía al microservicio en Python, que transforma el texto en un vector numérico mediante el modelo paraphrase-multilingual-mpnet-base-v2.
Ese vector se compara con los vectores almacenados en la base de datos MySQL, en la tabla conocimiento, y se selecciona la respuesta con mayor similitud.

Si el resultado supera el umbral definido en el archivo app.config.json, el sistema devuelve directamente la mejor coincidencia; de lo contrario, ofrece un conjunto de respuestas relacionadas.
Cada respuesta incluye la fuente o el fragmento correspondiente, y el usuario puede indicar si fue útil (pulgar arriba o pulgar abajo).
Esa valoración se almacena en la tabla retroalimentacion de la base de datos museo, donde queda registrada junto con la fecha, la pregunta y la respuesta.

Todo el procesamiento se realiza de forma local y sin conexión a Internet, utilizando únicamente los recursos del servidor donde se ejecuta el sistema.

La documentación técnica de instalación y despliegue se encuentra en el archivo README_produccion.md dentro del mismo repositorio:
